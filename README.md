# Academic Plagiarism Analyzer

A reference-aware plagiarism detection system comparable in rigor to commercial solutions like Turnitin.

## Features

- **DOI-based analysis**: Compares text only against cited references
- **Multi-modal similarity**: Combines lexical and semantic analysis
- **Academic rigor**: Turnitin-compatible classification thresholds
- **Open access preference**: Ethical sourcing of reference material
- **Comprehensive reporting**: PDF, CSV, visualization outputs
- **Batch processing**: Analyze multiple documents efficiently
- **Caching system**: Ensures reproducibility and performance
